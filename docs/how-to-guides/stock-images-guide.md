# Stock Images Guide for Band of Bakers

## Overview

This guide provides curated stock image recommendations for all areas of the Band of Bakers website. All images should maintain a consistent, professional, artisan aesthetic that reflects the traditional, family-owned bakery brand.

## üéØ Image Style Guidelines

**Photography Style:**

- Warm, natural lighting
- Rustic, handmade aesthetic
- Professional food photography
- Earth tones and warm color palette
- Authentic, not overly styled
- Focus on texture and detail

**Technical Specifications:**

- High resolution (minimum 1920px wide for hero images)
- Web-optimized formats (WebP preferred, JPEG fallback)
- Consistent aspect ratios where possible
- Alt text friendly (describe the image clearly)

## üé® Implementation Notes

### Image Optimization

1. **Format:** Use WebP with JPEG fallback
2. **Compression:** Optimize for web (80-85% quality)
3. **Sizes:** Provide multiple sizes for responsive design:
   - Thumbnail: 300x300px
   - Medium: 600x600px
   - Large: 1200x1200px
   - Hero: 2000x1200px

### Lazy Loading

- Implement lazy loading for all product images
- Use blur-up technique for better perceived performance

### Accessibility

- Always include descriptive alt text
- Ensure sufficient color contrast
- Provide text alternatives where needed

### CDN Recommendations

Consider using a CDN for better performance:

- **Unsplash:** Direct integration available
- **Cloudinary:** Automatic optimization and transformation
- **Imgix:** Advanced image processing

## üí° Pro Tips

1. **Seasonal Rotation:** Consider updating some images seasonally (Christmas bakes, summer fruits, etc.)
2. **Local Touch:** Mix in photos of actual bakery (when available) with stock images
3. **Consistency:** Maintain the same filter/lighting style across all images
4. **Mobile-First:** Test all images on mobile devices for clarity
5. **Loading Speed:** Implement responsive images with `srcset` for optimal loading
